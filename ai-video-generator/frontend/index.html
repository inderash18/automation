Missing API Credentials (client_secrets.json):
To publish directly from the dashboard, you must download desktop credentials:
  1. Go to the Google Cloud Console, create a project, and enable the YouTube Data API v3.
  2. Configure the OAuth Consent Screen (External/Internal, add your email as a test user).
  3. Go to Credentials -> Create Credentials -> OAuth client ID (Application Type: Desktop App).
  4. Download the JSON file, rename it to client_secrets.json, and save it in the root folder of this project.
